5个让你相见恨晚的 VS Code 插件

如果你每天都在用 VS Code 写代码，但却还没装这几个插件，那你真的错过了一个亿！

今天小白就给你安利 5 个让我相见恨晚的 VS Code 插件，装完之后，编码效率直接起飞 🚀

1. Prettier - 代码格式化神器

Prettier 是一个代码格式化工具，支持 JavaScript、TypeScript、CSS、HTML、JSON 等几乎所有主流语言。

安装方法

在 VS Code 扩展商店搜索 “Prettier - Code: formatter”，点击安装即可。

推荐配置

打开 VS Code 设置（Ctrl + ,），搜索 “format”，勾选以下选项：

  • ✅ Editor: Format On Save - 保存时自动格式化
  • ✅ Editor: Default Formatter - 选择 “Prettier - Code: formatter”

为什么推荐它？

让代码风格统一，让团队协作更顺畅！

2. ESLint - 代码质检员

ESLint 是一个静态代码分析工具，用来检查 JavaScript/TypeScript 代码中的潜在问题。

它能发现哪些问题？

  • 未使用的变量
  • 未定义的变量
  • 重复定义的变量
  • 语法不规范
  • 死循环、不可达代码

安装配置

  1. 安装 ESLint 扩展

    在 VS Code 扩展商店搜索 “ESLint”，安装由 Dirk Baeumer 开发的官方扩展。

最佳实践：ESLint 和 Prettier 一起用，前者查问题，后者管格式！

3. GitLens - Git 超能力

GitLens 是 VS Code 上最强大的 Git 增强插件，让你对代码历史的掌控达到新高度。

核心功能

  • 行内标注：每一行代码后面都会显示最后修改者、修改时间、提交信息
  • 代码追溯：查看某行代码的所有历史修改
  • 提交视图：图形化的分支历史

安装

搜索 “GitLens — Git supercharged” 并安装。

4. Live Server - 实时预览

Live Server 是前端开发必备插件，它可以让你在本地启动一个开发服务器，并且保存文件时自动刷新浏览器。

使用方法

  1. 打开你的 HTML 文件
  2. 点击右下角的 “Go Live” 按钮
  3. 浏览器会自动打开并显示你的页面
  4. 修改代码并保存，浏览器自动刷新

为什么需要它？

  • 告别手动刷新浏览器的痛苦
  • 实时预览修改效果
  • 提高开发效率 200%

5. Bracket Pair Colorizer - 括号高亮

Bracket Pair Colorizer（现在已内置到 VS Code）可以帮你用不同颜色标识配对的括号，让代码结构更清晰。

效果展示

每一对括号都有不同的颜色，嵌套关系一目了然！
